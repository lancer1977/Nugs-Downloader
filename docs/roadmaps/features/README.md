# Feature-Level Roadmaps

This folder groups short, feature-scoped roadmap surfaces.
Feature-level docs should stay close to user-visible behavior and usually map to one feature area or one compact operational surface.

## Current feature surfaces

- [C# Multisite Rewrite](../../features/csharp-multisite-rewrite.md)
- [Portainer Stack](../../features/portainer-stack/README.md)

## Use this folder for

- compact feature landing pages
- feature checklists that stay tied to one visible surface
- narrow operational docs that need their own roadmap-style tracking

## Keep out of this folder

- broad multi-phase initiative trackers
- cross-repo programs
- portfolio-wide prioritization docs
